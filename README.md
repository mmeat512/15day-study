# 📈 StockStudy-15Day-Tracker

## 🚀 프로젝트 소개

**StockStudy-15Day-Tracker**는 책 기반의 15일 과정 주식 스터디를 체계적으로 관리하고, 스터디 참여자들의 학습 진도와 개인 경험을 효율적으로 기록하고 공유하기 위한 웹 애플리케이션입니다.

개개인의 **과제 제출, 학습 느낀점, 소감** 등을 기록하여 15일 동안의 학습 몰입도와 완주율을 높이는 것을 목표로 합니다.

## ✨ 주요 기능

이 프로젝트는 다음과 같은 핵심 기능을 제공합니다.

### 1. 🗓️ 15일 진도 관리 시스템

* **Day별 일정 등록:** Day 1부터 Day 15까지 책의 챕터/과제에 맞춘 학습 내용을 등록하고 관리합니다.
* **진도율 표시:** 개인 및 스터디 그룹 전체의 현재 진행 상황을 백분율(%)로 시각화하여 제공합니다.

### 2. ✅ 과제 제출 및 경험 기록

* **과제 제출 폼:** Day별 지정된 과제 내용을 텍스트, 이미지, 파일 등으로 제출할 수 있습니다.
* **느낀점 / 소감 작성:** 학습 내용 요약 및 개인적인 성찰(느낀점, 소감)을 기록하며, 공개/비공개 설정을 지원합니다.
* **제출 현황 대시보드:** 본인의 과제 제출 여부와 멤버들의 전체 제출 현황을 한눈에 파악합니다.

### 3. 💬 피드백 및 커뮤니티

* 제출된 과제 및 느낀점에 대해 운영자 또는 다른 멤버들이 댓글을 달아 피드백을 공유할 수 있습니다.
