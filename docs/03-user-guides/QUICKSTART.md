# ⚡ 빠른 시작 가이드

> 5분 안에 Stock Study 시작하기!

## 🚀 3단계로 시작하기

### 1단계: 회원가입 (1분)

1. 브라우저를 열고 주소창에 입력:
   ```
   http://localhost:3090/register
   ```

2. 정보 입력:
   - 이메일 입력
   - 사용자명 만들기 (별명)
   - 비밀번호 만들기
   - 비밀번호 확인

3. "회원가입" 버튼 클릭!

### 2단계: 로그인 (30초)

1. 주소창에 입력:
   ```
   http://localhost:3090/login
   ```

2. 방금 만든 정보로 로그인:
   - 이메일 또는 사용자명 입력
   - 비밀번호 입력

3. "로그인" 버튼 클릭!

### 3단계: 스터디 시작 (2분)

**A. 새 스터디 만들기**
```
http://localhost:3090/studies/create
```
- 스터디 이름 입력
- 책 제목 입력
- 날짜 선택 (15일간)
- 최대 인원 설정
- "스터디 만들기" 클릭!
- 초대 코드를 친구들에게 공유!

**B. 친구 스터디 참여하기**
```
http://localhost:3090/studies/join
```
- 친구에게 받은 초대 코드 입력
- "참여하기" 클릭!

## ❓ 문제 해결

### 로그인이 안 돼요
- 이메일/사용자명 다시 확인
- 비밀번호 대소문자 확인
- 회원가입을 먼저 했는지 확인

### 페이지가 안 열려요
- 인터넷 연결 확인
- 브라우저 새로고침 (F5)
- 주소를 정확하게 입력했는지 확인

### 초대 코드가 안 맞아요
- 코드 대소문자 정확히 입력
- 친구에게 코드 다시 확인
- 공백 없이 입력
