## 🎯 이 앱은 뭐하는 거예요?

**Stock Study 15-Day Tracker**는 친구들과 함께 주식 공부를 15일 동안 하는 걸 도와주는 웹사이트예요!

### 🌟 이런 걸 할 수 있어요:

- 📝 **친구들과 스터디 그룹 만들기** - 같이 공부할 친구들을 모을 수 있어요
- 📅 **매일 과제 받기** - 15일 동안 매일 할 일을 알려줘요
- ✅ **과제 제출하기** - 공부한 내용을 올릴 수 있어요
- 📊 **진행 상황 보기** - 내가 얼마나 했는지 확인할 수 있어요
- 💬 **친구들 댓글 달기** - 서로 응원하고 도와줄 수 있어요

## 🔧 시작하기 전에

### 필요한 것들

1. **컴퓨터 또는 태블릿** - 인터넷이 되는 기기
2. **인터넷 연결** - 와이파이나 데이터
3. **웹브라우저** - 크롬, 사파리, 엣지 등

### 접속 방법

1. 웹브라우저(크롬, 사파리 등)를 열어요
2. 주소창에 `http://localhost:3090` 를 입력해요
3. 엔터를 눌러요!

## 📝 회원가입 하는 방법

처음 사용하는 사람은 계정을 만들어야 해요. 걱정하지 마세요, 정말 쉬워요!

### 단계별 따라하기

#### 1단계: 회원가입 페이지로 가기

```
http://localhost:3090/register
```

위 주소를 주소창에 입력하고 엔터를 눌러요.

#### 2단계: 정보 입력하기

화면에 빈칸이 나타날 거예요. 하나씩 채워볼까요?

- **📧 이메일 (Email)**: 내 이메일 주소를 적어요
  - 예시: `hongkildong@example.com`

- **👤 사용자명 (Username)**: 친구들에게 보일 별명이에요
  - 예시: `주식왕홍길동`
  - 특별한 점: 이 별명은 세상에 하나뿐이어야 해요!

- **🔒 비밀번호 (Password)**: 비밀번호를 만들어요
  - 다른 사람이 모르는 걸로!
  - 최소 6글자 이상이면 좋아요

- **🔒 비밀번호 확인**: 위에 쓴 비밀번호를 똑같이 한번 더 써요
  - 틀리지 않았는지 확인하는 거예요

#### 3단계: 회원가입 버튼 누르기

모든 칸을 채웠으면 **"회원가입"** 버튼을 눌러요!

#### 4단계: 완료!

✅ 성공하면 로그인 페이지로 자동으로 이동해요.

### 💡 팁

- 이메일은 나중에 비밀번호를 잊어버렸을 때 필요해요
- 사용자명은 나중에 바꿀 수 없으니 신중하게 정해요
- 비밀번호는 꼭 기억해야 해요! 메모해두면 좋아요

## 🔑 로그인 하는 방법

계정을 만들었으면 이제 로그인할 차례예요!

### 단계별 따라하기

#### 1단계: 로그인 페이지로 가기

```
http://localhost:3090/login
```

위 주소를 주소창에 입력하고 엔터를 눌러요.

#### 2단계: 정보 입력하기

두 가지 방법으로 로그인할 수 있어요:

**방법 1: 이메일로 로그인**
- 📧 이메일 칸: 가입할 때 쓴 이메일을 입력해요
- 🔒 비밀번호 칸: 가입할 때 만든 비밀번호를 입력해요

**방법 2: 사용자명으로 로그인**
- 👤 사용자명 칸: 가입할 때 만든 별명을 입력해요
- 🔒 비밀번호 칸: 가입할 때 만든 비밀번호를 입력해요

#### 3단계: 로그인 버튼 누르기

정보를 입력했으면 **"로그인"** 버튼을 눌러요!

#### 4단계: 완료!

✅ 성공하면 대시보드 페이지로 자동으로 이동해요.

### ⚠️ 문제가 생겼을 때

- **"이메일 또는 비밀번호가 틀렸어요"** 라고 나오면?
  - 이메일이나 사용자명을 정확하게 입력했는지 확인해요
  - 비밀번호를 다시 확인해요 (대문자/소문자 주의!)

- **화면이 안 바뀌면?**
  - 인터넷 연결을 확인해요
  - 버튼을 다시 한번 눌러봐요

## 🏠 대시보드 사용하기

로그인하면 제일 먼저 보이는 화면이 **대시보드**예요.
여기서 오늘 할 일과 진행 상황을 한눈에 볼 수 있어요!

### 화면 구성

```
http://localhost:3090/dashboard
```

대시보드는 크게 세 부분으로 나뉘어요:

#### 1. 👋 환영 메시지

```
Dashboard
Welcome back, [내 사용자명]
```

- 내 이름이 보여요
- 로그인이 잘 됐는지 확인할 수 있어요

#### 2. 📊 스터디 개요 카드

왼쪽 큰 카드에서 볼 수 있는 것들:

- **스터디 이름**: 지금 참여 중인 스터디 이름
  - 예: "January Stock Study"

- **책 제목**: 함께 공부하는 책
  - 예: "Stock Investment for Beginners"

- **날짜**: 언제 시작해서 언제 끝나는지
  - 예: 2025년 1월 15일 ~ 1월 29일

- **진행률**: 얼마나 했는지 퍼센트로 표시
  - 예: 33% (5일째 / 총 15일)

- **초대 코드**: 친구들을 초대할 수 있는 비밀 번호
  - 예: ABC12345

#### 3. 📈 내 통계

오른쪽 카드에서 볼 수 있는 것들:

- **Total Submissions (총 제출 횟수)**: 내가 과제를 몇 번 냈는지
- **Comments Received (받은 댓글 수)**: 친구들이 내 과제에 댓글을 몇 개 달았는지

#### 4. 📝 오늘의 과제

아래쪽에 오늘 해야 할 과제가 나와요:

- **Day 5**: 오늘이 5일째라는 뜻
- **제목**: 오늘 공부할 주제
  - 예: "Financial Statements Analysis"
- **학습 목표**: 오늘 배울 내용
  - 예: "재무제표의 기초 이해하기"
- **챕터**: 책의 몇 장을 읽어야 하는지
  - 예: "Chapter 3"

### 💡 대시보드에서 할 수 있는 일

- **View Details (자세히 보기)** 버튼: 스터디 상세 페이지로 이동
- **Start Assignment (과제 시작하기)** 버튼: 오늘의 과제 페이지로 이동

## 📚 스터디 만들기

친구들과 함께 공부할 새로운 스터디 그룹을 만들어볼까요?

### 단계별 따라하기

#### 1단계: 스터디 생성 페이지로 가기

```
http://localhost:3090/studies/create
```

위 주소를 입력하거나, 메뉴에서 "Create Study (스터디 만들기)"를 클릭해요.

#### 2단계: 스터디 정보 입력하기

화면에 나오는 빈칸들을 하나씩 채워볼까요?

##### 📌 기본 정보

- **스터디 이름 (Study Name)**
  - 예: "겨울방학 주식 스터디"
  - 친구들이 알아볼 수 있는 이름으로!

- **설명 (Description)** [선택사항]
  - 예: "주식 초보자들을 위한 기초 스터디"
  - 이 스터디가 무엇을 하는지 간단히 설명해요

- **책 제목 (Book Title)**
  - 예: "주식투자 무작정 따라하기"
  - 같이 공부할 책 이름을 써요

##### 📅 기간 설정

- **시작 날짜 (Start Date)**
  - 예: 2025년 1월 15일
  - 스터디를 시작할 날짜를 골라요

- **종료 날짜 (End Date)**
  - 예: 2025년 1월 29일
  - 스터디가 끝나는 날짜를 골라요
  - 💡 팁: 시작 날짜로부터 15일 뒤로 설정하면 딱 맞아요!

##### 👥 참여 인원

- **최대 인원 (Max Members)**
  - 예: 10명
  - 몇 명까지 참여할 수 있는지 정해요
  - 최소 2명 ~ 최대 50명까지 가능해요

#### 3단계: 스터디 만들기 버튼 누르기

모든 정보를 입력했으면 **"Create Study (스터디 만들기)"** 버튼을 눌러요!

#### 4단계: 초대 코드 받기

✅ 성공하면:
- 스터디가 만들어져요!
- **초대 코드**를 받아요 (예: ABC12345)
- 이 코드를 친구들에게 알려주세요!

### 💡 스터디 만들기 팁

- **이름은 간단하고 명확하게**: 너무 긴 이름은 기억하기 어려워요
- **날짜는 현실적으로**: 15일 동안 꾸준히 할 수 있는 날짜를 선택해요
- **인원은 적당하게**: 너무 많으면 관리가 어려워요. 5~10명 정도가 좋아요

## 🤝 스터디에 참여하기

친구가 만든 스터디에 참여하고 싶나요? 초대 코드만 있으면 쉽게 참여할 수 있어요!

### 단계별 따라하기

#### 1단계: 스터디 참여 페이지로 가기

```
http://localhost:3090/studies/join
```

위 주소를 입력하거나, 메뉴에서 "Join Study (스터디 참여하기)"를 클릭해요.

#### 2단계: 초대 코드 입력하기

- **Invite Code (초대 코드)** 칸에 친구에게 받은 코드를 입력해요
  - 예: ABC12345
  - 대문자/소문자를 정확하게 입력해야 해요!

#### 3단계: 참여하기 버튼 누르기

**"Join Study (스터디 참여하기)"** 버튼을 눌러요!

#### 4단계: 완료!

✅ 성공하면:
- 스터디 멤버가 돼요!
- 대시보드에서 스터디 정보를 볼 수 있어요
- 오늘의 과제를 확인할 수 있어요

### ⚠️ 문제가 생겼을 때

- **"초대 코드가 틀렸어요"** 라고 나오면?
  - 코드를 다시 확인해요
  - 친구에게 코드가 맞는지 물어봐요

- **"인원이 다 찼어요"** 라고 나오면?
  - 스터디가 가득 찬 상태예요
  - 다른 스터디를 찾아보거나 새로 만들어요

## 👤 마이페이지 확인하기

내 프로필과 활동 내역을 확인할 수 있는 곳이에요!

### 단계별 따라하기

#### 1단계: 마이페이지로 가기

```
http://localhost:3090/mypage
```

위 주소를 입력하거나, 메뉴에서 "My Page (마이페이지)"를 클릭해요.

#### 2단계: 내 정보 확인하기

마이페이지에서 볼 수 있는 것들:

##### 👤 프로필 카드

- **사용자명**: 내 별명
- **이메일**: 가입할 때 쓴 이메일
- **가입일**: 언제 가입했는지

##### 📊 통계 카드

- **참여 중인 스터디**: 지금 하고 있는 스터디 개수

- **받은 댓글**: 친구들이 준 댓글 개수
- **작성한 댓글**: 내가 친구들에게 쓴 댓글 개수

##### 📚 내 스터디 목록

- 참여하고 있는 모든 스터디 목록이 나와요
- 각 스터디의 진행률도 볼 수 있어요
- 클릭하면 해당 스터디로 이동해요

## ❓ 자주 묻는 질문

### Q1: 비밀번호를 잊어버렸어요!

**A:** 로그인 페이지에서 "비밀번호 찾기" 버튼을 눌러요. 가입할 때 쓴 이메일로 새 비밀번호를 만들 수 있는 링크가 보내져요.

### Q2: 사용자명을 바꾸고 싶어요

**A:** 죄송해요, 한번 정한 사용자명은 바꿀 수 없어요. 새로운 계정을 만들어야 해요.

### Q3: 스터디를 중간에 그만둘 수 있나요?

**A:** 네! 스터디 페이지에서 "Leave Study (스터디 나가기)" 버튼을 누르면 돼요.

### Q4: 과제를 놓쳤어요. 나중에 해도 되나요?

**A:** 네, 가능해요! 이전 날짜의 과제도 언제든 할 수 있어요. 하지만 진행률에는 영향을 줄 수 있어요.

### Q5: 친구 찾기 기능이 있나요?

**A:** 지금은 초대 코드로만 친구를 초대할 수 있어요. 친구에게 코드를 직접 알려주세요!

### Q6: 스터디를 만들었는데 정보를 수정하고 싶어요

**A:** 스터디를 만든 사람(Owner)은 스터디 설정에서 정보를 수정할 수 있어요.

### Q7: 하루에 여러 개의 스터디에 참여할 수 있나요?

**A:** 네! 원하는 만큼 많은 스터디에 참여할 수 있어요.

### Q8: 앱이 느리거나 안 열려요

**A:**
1. 인터넷 연결을 확인해요
2. 브라우저를 새로고침 해요 (F5 키 또는 ⌘+R)
3. 브라우저를 닫았다가 다시 열어봐요

## 🎯 성공적인 스터디를 위한 팁

### ✨ 공부 팁

1. **매일 조금씩**: 한꺼번에 하기보다 매일 조금씩 하는 게 좋아요
2. **시간 정하기**: 매일 같은 시간에 하면 습관이 돼요
3. **친구와 함께**: 혼자보다 친구들과 하면 더 재미있어요

### 💪 스터디 팀 관리 팁

1. **규칙 정하기**: 시작하기 전에 간단한 규칙을 정해요
2. **응원하기**: 친구들 과제에 좋은 댓글을 달아주세요
